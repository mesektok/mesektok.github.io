GitHub Pages로 블로그를 만들 때 Chirpy 테마는 꽤 깔끔한 선택지입니다.

사이드바 구조와 포스트 화면이 잘 정리되어 있어서 기술 블로그나 기록형 블로그에 잘 어울립니다.

다만 처음 적용할 때는 설정 파일과 폴더 구조를 정확히 이해해야 시행착오를 줄일 수 있습니다.

이 글에서는 Chirpy를 적용하면서 실제로 체크했던 순서를 정리해 보겠습니다.
