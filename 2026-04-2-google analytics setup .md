블로그를 만들고 나서 가장 먼저 하고 싶었던 것이 있습니다.

**"과연 누가 내 블로그를 보고 있을까?"**

Google Analytics를 연결하면 방문자 수, 어떤 글을 봤는지,  
어느 나라에서 왔는지까지 전부 확인할 수 있습니다.

애드센스 심사에도 유리합니다.  
지금 바로 연결해 두는 것을 추천합니다.

---

## 전체 순서

```
① Google Analytics 계정 만들기
② 측정 ID 복사
③ _config.yml 에 붙여넣기
④ 데이터 수집 확인
```

---

## 1단계 — Google Analytics 접속

```
analytics.google.com 접속
→ 구글 계정으로 로그인
→ "측정 시작" 클릭
```

> 이미 Gmail이 있으면 별도 가입 없이 바로 로그인 가능합니다.

---

## 2단계 — 계정 만들기

```
계정 이름 입력
→ mesektok  (본인 블로그명)
→ "다음" 클릭
```

---

## 3단계 — 속성 만들기

```
속성 이름: mesektok.com
보고 시간대: 대한민국
통화: 원(₩)
→ "다음" 클릭
```

---

## 4단계 — 비즈니스 정보

```
업종 카테고리: 기타
비즈니스 규모: 소규모
→ "다음" 클릭
→ 이용약관 동의 → "만들기" 클릭
```

---

## 5단계 — 웹 스트림 설정

```
플랫폼 선택 화면에서 "웹" 클릭

웹사이트 URL: mesektok.com
스트림 이름: mesektok 블로그
→ "스트림 만들기" 클릭
```

---

## 6단계 — 측정 ID 복사

스트림 세부정보 화면에서:

```
측정 ID 확인
예시: G-XXXXXXXXXX
     ↑ 이 형식입니다

오른쪽 복사 아이콘 클릭
```

> **G- 로 시작하는 코드 전체를 복사합니다.**

---

## 7단계 — `_config.yml` 수정

GitHub 저장소에서 `_config.yml` 열기 → ✏️ 클릭

아래 부분을 찾아서 수정합니다:

```yaml
# 수정 전
analytics:
  google:
    id: # fill in your Google Analytics ID

# 수정 후
analytics:
  google:
    id: G-XXXXXXXXXX
```

`G-XXXXXXXXXX` 자리에 복사한 측정 ID 붙여넣기

→ 맨 아래 **"Commit changes"** 클릭

---

## 8단계 — 연결 확인

```
① 2~3분 대기 (GitHub 배포 시간)

② mesektok.com 접속
   (본인이 직접 방문)

③ Google Analytics 로 돌아가서
   → 왼쪽 메뉴 "보고서" → "실시간"

④ 활성 사용자 1명 표시되면 완료 ✅
```

---

## 자주 하는 실수

| 실수 | 해결 |
|---|---|
| `UA-` 로 시작하는 코드 입력 | 구버전 코드입니다. `G-` 로 시작하는 코드를 사용하세요 |
| 코드 앞뒤에 따옴표 추가 | 따옴표 없이 그대로 입력 |
| Commit 후 바로 확인 | 2~3분 대기 후 확인 |
| 실시간에 0명 표시 | 본인이 직접 블로그 접속 후 다시 확인 |

---

## 연결 후 확인할 것들

연결 첫날은 실시간 방문자 확인만 됩니다.  
아래 데이터는 시간이 쌓이면 자동으로 보입니다.

| 데이터 | 확인 가능 시점 |
|---|---|
| 실시간 방문자 | 즉시 |
| 일별 방문자 수 | 다음날부터 |
| 어떤 글을 봤는지 | 2~3일 후 |
| 방문자 유입 경로 | 1주일 후 |
| 국가별 방문자 | 1주일 후 |

---

## 마치며

Google Analytics 연결은 블로그 운영의 기본 중 기본입니다.

애드센스 신청 전에 **최소 2~4주** 데이터가 쌓여있으면  
심사에 훨씬 유리하다고 알려져 있습니다.

지금 바로 연결해 두세요.
