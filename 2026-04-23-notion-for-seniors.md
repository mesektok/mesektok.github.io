처음 Notion이라는 단어를 들었을 때  
또 새로운 앱을 배워야 하나 싶었습니다.

메모앱, 카카오톡 나에게 보내기, 포스트잇으로도  
충분히 살아왔는데 말이죠.

그런데 써보니 달랐습니다.  
**이건 진짜 쓸 만합니다.**

---

## Notion 이 뭔가요?

한 줄로 말하면:

> **메모장 + 표 + 할일 관리 + 데이터베이스**가 하나로 합쳐진 무료 도구

PC, 태블릿, 스마트폰 어디서든 열립니다.  
인터넷만 되면 됩니다.

---

## 시니어에게 맞는 이유

| 특징 | 시니어에게 좋은 점 |
|---|---|
| 무료 | 부담 없이 시작 가능 |
| 클라우드 저장 | 폰에서 쓴 것이 PC에 그대로 |
| 한글 완벽 지원 | 불편함 없음 |
| 설치 불필요 | 브라우저에서 바로 사용 |
| 구글 계정 연동 | 별도 가입 없이 시작 |

---

## 시작하는 방법

```
① notion.so 접속
② "무료로 시작하기" 클릭
③ 구글 계정으로 로그인
④ 완료
```

그게 전부입니다.

---

## 제가 실제로 쓰는 방법 3가지

### 1. 블로그 글감 관리

가장 많이 씁니다.

떠오르는 글 아이디어를 표에 저장합니다.

| 제목 아이디어 | 상태 | 메모 |
|---|---|---|
| Git 처음 설치한 날 | 작성완료 | 삽질 3시간 |
| 애드센스 신청 후기 | 작성중 | 반려 대비 내용 추가 |
| Notion 쓰는 이유 | 아이디어 | 이 글 |

산책 중에 떠오른 아이디어를  
스마트폰 Notion에 바로 입력합니다.  
PC에서 열면 그대로 있습니다.

### 2. 일정 관리

블로그 발행 일정을 달력으로 관리합니다.

```
새 페이지 → Calendar 선택
→ 날짜별로 포스팅 제목 입력
→ 한눈에 계획이 보입니다
```

### 3. 정보 수집

유용한 링크, 참고 자료를  
주제별로 페이지를 만들어 저장합니다.

```
애드센스 관련
├── 신청 조건 정리
├── 유용한 링크 모음
└── 참고한 블로그들
```

---

## 처음 배울 때 알아두면 좋은 것

### 블록 개념

Notion 의 모든 것은 **블록**으로 이루어져 있습니다.  
글 한 줄, 표 하나, 이미지 하나가 전부 블록입니다.

```
새 줄에서 / 를 입력하면
→ 블록 종류 선택 메뉴가 나옵니다

자주 쓰는 것:
/table   → 표 만들기
/page    → 새 페이지 만들기
/todo    → 할일 목록
/h1      → 큰 제목
/h2      → 중간 제목
```

### 페이지 안에 페이지

Notion 은 페이지 안에 페이지를 만들 수 있습니다.

```
블로그 관리 (메인 페이지)
├── 글감 목록
├── 발행 일정
├── 참고 자료
└── 수익 기록
```

폴더처럼 정리하면 됩니다.

## 처음에 헷갈렸던 것들

**Q. 저장 버튼이 없는데 괜찮나요?**  
자동 저장됩니다. 입력하는 즉시 저장됩니다.

**Q. 실수로 지웠을 때는요?**  
`Ctrl + Z` 로 되돌리기 가능합니다.  
또는 오른쪽 상단 `...` → 페이지 기록에서 복구 가능합니다.

**Q. 오프라인에서도 되나요?**  
인터넷이 필요합니다. 오프라인은 지원이 제한적입니다.

**Q. 유료로 전환해야 하나요?**  
무료로 충분합니다. 저는 무료 플랜만 씁니다.

---

## 시니어에게 추천하는 시작 순서

```
1주차: 글감 수집 표 하나만 만들기
2주차: 스마트폰 앱 설치 → 이동 중 아이디어 입력
3주차: 블로그 발행 일정 달력 추가
4주차: 참고 자료 페이지 만들기
```

한 번에 다 배우려 하지 않습니다.  
**딱 하나씩, 필요할 때 하나씩** 추가하면 됩니다.
